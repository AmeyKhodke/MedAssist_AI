## Data Flow

1. **User Interaction:** User interacts with the React frontend (e.g., sends a chat message requesting a medicine).
2. **API Request:** Frontend sends an HTTP POST request via Axios to the FastAPI backend (e.g., `/agent/chat`).
3. **Agent Processing:** Backend routes request to the `agents.py` module. It saves the message, extracts intent, checks safety, and potentially executes the order or prompts for confirmation.
4. **Data Persistence:** `agents.py` and `app.py` interact with `database.py` to deduct stock, record the order, or log chat history in the SQLite database.
5. **Response:** Backend returns the result/status to the frontend, which updates the UI. Traces are sent asynchronously to Langfuse.
